# 📊 SQL Business Sales Analysis Project

## Project Overview

This project demonstrates how SQL can be used to perform business analysis on a retail sales dataset.

The objective is to answer important business questions related to revenue, profit, products, customers, and sales performance. Each analysis is performed using SQL queries and interpreted from a business perspective.

---

# Business Objectives

This project answers the following questions:

1. What is the total revenue generated?
2. What is the total profit earned?
3. Which are the top 10 selling products?
4. Which product categories generate the highest revenue?
5. Which countries contribute the most revenue?
6. How do monthly sales change over time?
7. Which gender purchases more?
8. Which age group generates the highest sales?
9. Which product is the most profitable?
10. What is the overall profit margin?
